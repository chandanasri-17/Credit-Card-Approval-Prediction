# 💳 Credit Card Approval Prediction using Machine Learning

A Machine Learning-based web application that predicts whether a credit card application is likely to be **Approved** or **Rejected** based on applicant information. The project automates the credit card approval process using a trained Random Forest model and provides a user-friendly web interface developed with Flask.

---

## 📌 Project Overview

Financial institutions receive numerous credit card applications every day. Manually evaluating each application can be time-consuming and prone to inconsistencies. This project leverages Machine Learning to assist in the decision-making process by predicting the approval status of an application based on applicant details.

The application provides a simple interface where users can enter applicant information, and the trained model instantly predicts whether the application is likely to be approved or rejected.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/chandanasri-17/Credit-Card-Approval-Prediction.git
```

### Navigate to the Source Code

```bash
cd "Project Development Phase/Credit Card Approval System"
```

### Create a Virtual Environment (Optional)

```bash
python -m venv venv
```

### Activate the Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Flask Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```

## 📊 Project Workflow

1. User enters applicant details.
2. Input data is validated.
3. Data is preprocessed.
4. Random Forest model analyzes the applicant information.
5. Prediction is generated.
6. Result is displayed to the user.
